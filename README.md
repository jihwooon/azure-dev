# SpringBoot Example

이는 Spring Boot 애플리케이션용 REST API입니다.

## Tech Stack
- SpringBoot 3.2.3  
- Java zulu 17.0.9
- Gradle 8.4
- JPA
- H2 Database

## Getting Started
- Repository 복제합니다.
- Ide 에서 프로젝트를 열고 Maven Dependencies를 다운로드 받습니다.
- `application.yaml` 파일을 열고 `spring.datasource.url`을 수정합니다.
- Test를 실행합니다.
- 데이터베이스에 데이터가 추가 여부를 확인합니다.
- Spring Boot를 실행합니다.
